\doxysubsection{Detailed Description}
The main window of the University Examination Identity Verification System. 

This class represents the initial page of the application, including\+:
\begin{DoxyItemize}
\item A login screen for TAs or instructors to authenticate using credentials.
\item A dashboard that displays exam details and offers access to various exam-\/related operations. 
\end{DoxyItemize}
