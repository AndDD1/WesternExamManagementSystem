\doxysubsection{Detailed Description}
Implements the \doxylink{classSubmissionDialog}{Submission\+Dialog} class. 

This file defines the \doxylink{classSubmissionDialog}{Submission\+Dialog} class which provides a popup dialog for exam submission. It prompts the user to enter their student number, and logs the exam submission if the current time is within the exam period. If the current time is outside the exam period, the submit button is disabled.
