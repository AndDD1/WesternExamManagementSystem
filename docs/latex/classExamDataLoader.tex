\doxysubsection{Detailed Description}
Utility class for loading exam data from a structured text file into an \doxylink{classExam}{Exam} instance. 

This class provides a static method that parses exam details, proctor info, and student info from a given file and populates the corresponding fields in the \doxylink{classExam}{Exam} object.
