\doxysubsection{Detailed Description}
A Qt dialog for handling student check-\/in during the exam session. 

This dialog allows TAs or proctors to input a student ID, view student details, and confirm their attendance. It displays information such as name, date of birth, seat assignment, exam version, and student photo.
