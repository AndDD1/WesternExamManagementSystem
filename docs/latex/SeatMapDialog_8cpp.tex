\doxysubsection{Detailed Description}
Implementation of the \doxylink{classSeatMapDialog}{Seat\+Map\+Dialog} class. 

Dynamically creates a grid of buttons for each seat in the exam room using the max\+Row and max\+Col from the \doxylink{classExam}{Exam} object. Each button represents a seat in the room. When clicked, the system shows the student assigned to that seat and their details.
